# 🚀 OfertMatch

Plataforma inteligente de ofertas e promoções que conecta consumidores e empresas, facilitando a divulgação de produtos e a busca pelos melhores preços.

---

## 📌 Sobre o Projeto

O OfertMatch foi desenvolvido com foco em criar uma experiência moderna para divulgação de ofertas, permitindo que empresas publiquem promoções e usuários encontrem produtos com facilidade.

O projeto surgiu como uma iniciativa acadêmica com visão de evolução para uma plataforma escalável e colaborativa.

---

- CSS3

### Back-end
- Python
- Flask

- MongoDB Atlas

- Cadastro de ofertas

- Estrutura modular
